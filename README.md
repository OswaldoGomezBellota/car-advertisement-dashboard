# car-advertisement-dashboard

Dashboard web para análisis de datos de anuncios de vehículos

## Descripción del proyecto

El proyecto de desarrollo de una aplicación web interactiva para el análisis de datos de anuncios de venta de vehículos. La aplicación permite explorar el dataset mediante visualizaciones interactivas que ayudan a comprender mejor las características de los vehículos y sus precios.

El objetivo principal es practicar el uso de herramientas de desarrollo de software y construir un dashboard web utilizando Python.

## Funcionalidades de la aplicación
La aplicación permite a los usuarios:

- Visualizar una vista previa del dataset de vehículos.
- Generar un **histograma del kilometraje (odometer)** para observar la distribución del uso de los vehículos.
- Crear un **gráfico de dispersión entre el precio y el kilometraje**, lo que permite analizar la relación entre estas variables.
- Interactuar con los gráficos mediante botones o casillas de verificación dentro del dashboard.

## Tecnologías utilizadas
Este proyecto fue desarrollado utilizando:
- **Python**
- **Pandas** para manipulación de datos
- **Plotly Express** para visualizaciones interactivas
- **Streamlit** para crear la aplicación web
- **Render** para el despliegue en la nube
- **Git y GitHub** para control de versiones
